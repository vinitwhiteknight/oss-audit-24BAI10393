## Project Overview
This repository contains five shell scripts written for the VITyarthi Open Source Software Capstone Project. These scripts demonstrate practical Linux command-line skills, system administration, and automation concepts related to the open-source ecosystem.

## System Dependencies
To run these scripts successfully, your system requires:
* A Linux-based operating system (Tested on Ubuntu 24.04).
* The `bash` shell environment.
* Standard Linux core utilities (such as `grep`, `awk`, `du`, `uname`, and `whoami`).
* **For Script 2:** The `mysql-server` package should be installed via `apt` to see a positive result.
