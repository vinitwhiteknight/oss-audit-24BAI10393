## Project Overview
This repository contains the practical command-line deliverables for my Open Source Software Capstone Project. The broader project is a comprehensive audit of MySQL, exploring its origins, the ethical implications of the GPL v2 license, its footprint within a Linux environment, and its role in the LAMP stack. 

The five shell scripts hosted in this repository are designed to demonstrate practical Linux system administration, text processing, and automation skills.

## Software Profile: MySQL
As part of this audit, I evaluated MySQL against proprietary alternatives like Oracle Database. 
* **License:** GNU General Public License version 2 (GPL v2)
* **Architecture:** Relational Database Management System (RDBMS)
* **Ecosystem Role:** The foundational data layer (the "M") in the LAMP web stack.
